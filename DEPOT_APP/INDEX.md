## Description

CassKai est une plateforme SaaS de gestion financiere tout-en-un destinee aux PME et independants.
Elle propose 19 modules fonctionnels couvrant la comptabilite, la facturation, les banques, la fiscalite,
le CRM, les achats, le stock, les projets, les RH, et bien plus.

## Technologies principales

* Frontend : React 18, TypeScript, Vite, Tailwind CSS
* Backend : Supabase (PostgreSQL, Auth, Storage)
* Hebergement : Hostinger VPS avec Nginx
* Paiements : Stripe

## Conformite

* RGPD : Export donnees, droit a l'oubli, chiffrement AES-256
* LCEN : Mentions legales completes
* Normes comptables : PCG, SYSCOHADA, IFRS
* FEC : Fichier des Ecritures Comptables

## Installation a partir de ce depot

1. Extraire casskai-app-v1.0.0.zip
2. Installer les dependances : npm install
3. Configurer l'environnement : copier .env.example vers .env.local
4. Lancer en dev : npm run dev
5. Build production : npm run build

Voir README.md dans 2_Documentation pour plus de details.
